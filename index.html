Kalimat Pembuka

Hai, mungkin kamu (aku) buka lagi file keramat ini karena disini aku akan menceritakan "Cerita Memulai Coding". Aku adalah kamu di masa lalu yang membuat catatan digital menggunakan VS Code karena akan mendokumentasikan jejak coding ku secara digital. Oh iya, ini yg ngetik web itu aku (kamu) mungkin file ini akan menjadi kenangan dan akan selalu dikenang sama aku (kamu) kecuali semua data udh kehapus dan lupa nyimpan file ini. Aku bercerita pada tanggal 20 juni 2023 dimana aku baru tamat smp dan mendokumentasikan codingan ku. Aku baru saja lulus dan akibat gabut jadi menulis seluruh cerita ini. Okelah karena takut kepanjangan singkat cerita dimulailah awal dari seluruh cerita ini.

Bagian 1

Ini adalah bagian pertama dimana aku benaran pertama kali mulai belajar coding, oh iya, aku belajar coding karena hasutan akmal karena dia tertarik nyoba coding kemudian menghasutku untuk mengikutinya belajar coding. Aku awalnya tidak tertarik coding karena aku gk paham soal ilmu per-codingan sampai akhirnya dia menunjukkan sedikit hasil codingan dia, dan menunjukkan progress dia, karena coding itu bermain dengan code editor aku jadi tertarik untuk belajar, tapi belajar coding web/web developer. Oh iya, saat itu masih 2020 dimana pada saat itu adalah musim corona dan semuanya dilakukan secara daring (online), termasuk aku yang kerja nya cuma main game, pada saat itu aku baru masuk smp, sampailah dimana seluruh cerita hidupku yang membosankan (yang kerjanya main game terus) akhirnya memulai belajar coding, setelah 1 tahun coding aku stop belajar karena ngerasa gk ada gunanya dan saat itu pandemi sudah mulai mereda, jadi sekolah juga akhirnya dibuka dengan menggunakan sistem ganjil genap, jadi karena sekolah juga salah satu alasan kenapa aku stop coding.

Bagian 2

Setelah 1 tahun kemudian aku masuk kelas 9, cerita sekolah ku mulai berwarna dan aku pada saat itu menganggap bahwa kelas 9 adalah masa terbaik sekolah, sampai aku benar benar lupa tentang coding web, disini aku dk akan cerita tentang kehidupan sekolah di kelas 9 ku tapi aku akan menceritakan tentang "Memulai Coding Lagi", singkat cerita setelah 1 tahun kemudian (lagi) aku lanjut belajar coding dikarenakan sudah lulus smp dan libur sangat panjang. Cerita ini kubuat singkat aja karena sebentar lagi aku mau pergi ke Pontianak, jadi dulu saat aku stop coding bentar aku ada ngotak ngatik os di laptop, mulai dari linux kemudian mau mencoba menginstal macOS tapi gk berhasil sampai balik lagi ke windows dan pada saat aku menulis cerita ini aku menggunakan salah satu distro linux yaitu Pop OS. Mungkin aku dimasa depan tidak lagi menggunakan Pop OS dan kembali menggunakan windows, tapi aku mulai belajar coding saat aku menggunakan linux yaitu pop os dan ini adalah distro yang paling pas untuk ku. Semoga aku masih coding dimasa depan nanti dan semoga mendapatkan pekerjaan yang paling sesuai dengan minatnya.

Penutup

Ok segitu saja ceritaku untuk diri ku sendiri, aku menulis cerita ini karena ingin membuat catatan digital berupa web, dan mungkin diriku dimasa depan membaca ini atau setidaknya ingat kalau pernah menulis cerita tentang dirinya sendiri dan mungkin saja aku dimasa depan akan menganggap ini sebuah tindakan yang ga jelas dan dianggap cringe.
